# 解析WeChat的AccInfo.dat

## 介绍

**[仅支持Windows]**

本项目具备解析AccInfo.dat文件的功能，能够提取其中的wxid、用户名、邮箱、电话、头像URL以及其他未解析的数据。

您只需提供文件夹路径，剩下的任务就交给我来处理.
